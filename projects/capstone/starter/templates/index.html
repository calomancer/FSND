Introduction

What am I?

This is a sample page for an API I have created to help Table Top tournament organizers make, organize, and keep track of player characters and the groups of those characters.

Why am I?

This api was created for Organizers for Table Top Tournaments (Yeah, they have those) This allows an organizer, Game Masters, players, and those interested in the games to quickly create, edit, and organize player groups and characters related to those groups.

What do I do?

There are four ways to interact with this content.

  1. As Administrators
    1. View/Create/Edit/Delete Characters
    2. View/Create/Edit/Delete Groups
  2. As Dungeon/Game Masters
    1. View Characters
    2. View/Create/Edit/Delete Groups
  3. As Players
    1. View/Create/Edit/Delete Characters
    2. View Groups
  4. Unregistered users
    1. View Characters
    2. View Groups
